*&c  去访问c的地址用指针变量符 *     
& 提取地址
定义指针变量： int *pointer 此int为指针变量的基类型，是指针变量指向的变量的类型

int c = 76;
int *pointer;
pointer = &c;

pointer 所指向的存储单元的内容是变量c，而不是76. pointer可以当作是变量c。



cout << str + 5 << endl;
str+5指向','，并且是字符指针，对它使用cout，输出从str+5开始的字符串

a[4]
a   指向数组首元素的指针
&a  指向整个数组的指针

*E 如果E是一个指针，返回E所指向的内容
*(&a) == a
